# Release Notes - v1.0.1

**Release Date:** February 24, 2026

## 🎉 First Internal Release

v1.0.1 is the first internal release of `Skyline.DataMiner.SDM.Linq`, delivering LINQ query support for SDM repositories in DataMiner.

## ✨ Package Overview

This package enables `IQueryableRepository<T>` implementations so repository queries can be written with familiar C# LINQ syntax and translated into DataMiner-compatible filtering and sorting queries.

## 🚀 Core Capabilities

- Type-safe repository querying with standard LINQ expressions.
- Translation of query expressions to SDM/DataMiner filter and order operations.
- Support for common querying scenarios such as filtering, sorting, paging, and single-item retrieval.

## ✅ Supported LINQ Operations

- `Where` (including `&&`, `||`, `!`)
- `OrderBy`, `OrderByDescending`, `ThenBy`, `ThenByDescending`
- `Take`, `Skip`
- `First`, `FirstOrDefault`, `Single`, `SingleOrDefault`
- `Count`, `Any`

## 📚 Documentation

- Added a package README with installation and quick-start setup guidance.
- Included examples to onboard repository integrations faster.

## ✅ Summary

v1.0.1 establishes the initial public baseline of `Skyline.DataMiner.SDM.Linq` for practical LINQ-based querying on SDM repositories.

---

*This first release focuses on the core LINQ querying experience for SDM repositories; advanced projection and join/group scenarios are currently out of scope.*
